# Blockchain_Store
### Prerequisites

1.Download Ganache [https://www.trufflesuite.com/ganache](https://www.trufflesuite.com/ganache)
2.Open Ganache and click Quick Start for create your local Block Chain.

### Installation

1.Install NPM packages
   ```sh
   npm install
   ```
2.Compile truffle
   ```sh
   truffle compile
   ```
3.Migrate
   ```sh
   truffle migrate
   ```
4.Runing
   ```sh
   npm start
   ```
